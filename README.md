📌 Project Overview | Tổng Quan Dự Án

This project aims to provide a structured, easy-to-access database of Buddhist scriptures. By leveraging the Bilara format (segment-by-segment alignment), users can compare the Vietnamese translation directly with the English version and the original Pāli source.
Public Link : https://dhammaarchive.info/

Dự án này nhằm mục đích cung cấp một cơ sở dữ liệu kinh điển có cấu trúc, dễ tiếp cận. Bằng cách tận dụng định dạng Bilara (đối chiếu theo từng phân đoạn), người dùng có thể so sánh trực tiếp bản dịch tiếng Việt với bản tiếng Anh và nguồn gốc Pāli. 
Link truy cập: https://dhammaarchive.info/
